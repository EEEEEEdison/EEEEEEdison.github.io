1. Poultry Farming Intelligence System


Project Description:
This web application aims to help poultry farmers improve profitability by leveraging machine learning and big data techniques. For example, the application automates the HVAC settings based on indoor conditions. It can also predict certain poultry industry specific KPIs such as weight, mortality, feed conversion. Therefore, precision farming could be achieved which significantly reduces the cost of production.


• Machine Learning: Ridge Regression.
• Frontend: Node.JS.
• Backend: Django.
• Data pipeline: Apache Airflow.
• Library: Pandas, Numpy, Matplotlib, Seaborn, Sklearn, Mglearn.


Data Analysis: Broiler chicken weight VS Environmental factors




Network Diagram: overview of the technology infrastructure.



Machine Learning Pipeline: how each prediction models are trained using Apache AirFlow.




Project Outcome

This application intends to assist stakeholders in making informed business decisions driven by historical production data, ultimately enhancing production efficiency and profit margins. However, the first phase of this project faced challenges due to limited data availability, which could result in several issues such as overfitting and lack of diversity, lead to weak generalization to new data and imprecise predictions. Potential solutions have been proposed to industry partner, including data quality evaluation, data augmentation, and IoT data sharing frequency.




2. Driver Drowsiness Detection System


Project Description:
This project aims to detect driver drowsiness using computer vision and deep learning techniques. The application is able to detect the driver's eyes and mouth and alert the driver when the eyes are closed for a certain period of time. Thus potentially contribute to overall road safety.


• Machine Learning: Computer Vision.
• Algorithm: CNN (Convolutional Neural Network).
• Model: “ResNet-50” from TensorFlow.
• Library: Pandas, NumPy, Matplotlib, Tensorflow.keras, Scipy, etc.


Project idea: model can be embedded into future vehicle's interior camera.
